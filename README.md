# Iris Classification Perceptron

A simple machine learning project that implements a perceptron classification model using the Iris dataset.

## Files
- `iris_perceptron.ipynb` → Jupyter Notebook for training and testing the model
- `perceptron_model.py` → Python script version
- `perceptron_model.pkl` → saved trained model

## Technologies
- Python
- Scikit-learn
- Jupyter Notebook

## Description
This project was created to learn the basics of machine learning classification and perceptron algorithms using the Iris dataset.
